This repository contains the following algorithms/data structures:

Sorting :
* Selection sort
* Insertion sort
* Bubble sort
* Quick sort
* Merge sort
* Heap sorting (Binary heap)

Search :
* Binary array search
* Binary search tree

Graph :
* Breadth-First Search (BFS)
 - Algorithm that search the shortest path in unweighted graph
* Depth-First Search (DFS)
